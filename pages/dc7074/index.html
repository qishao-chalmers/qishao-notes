                mean_score = torch.ones_like(attn_weights, device=attn_weights.device)

                mean_score = attn_weights.sort().values[

                ]

                # compress start from start_compress_length
                for i in range(self.start_compress_length, src_len, 1):
                    window_start = i - self.lru_window if i - self.lru_window > 0 else 0

                    lru_score = (
                        attn_weights[:, window_start:i, :i]
                        > mean_score[:, window_start:i, :i]
                    ).sum(dim=1)

                    self.lru_keep_ratio += [
                        ((lru_score != 0).sum() / self.num_heads / (i + 1)).item()
                    ]

                    # keep last position_window tokens
                    positon_window = (
                        i - self.position_window if i - self.position_window > 0 else 0
                    )
                    lru_score[:, positon_window:] = (
                        1.0  # keep last position_window tokens
                    )

                    self.keep_ratio += [
                        ((lru_score != 0).sum() / self.num_heads / (i + 1)).item()
                    ]

                    # mask attention score
                    attn_weights[:, i, :i] = torch.where(
                        lru_score == 0,
                        torch.tensor(
                            torch.finfo(attn_weights.dtype).min,
                            device=attn_weights.device,
                        ).float(),
                        attn_weights_raw[:, i, :i].float(),
                    )
                    attn_weights[:, i, i:] = attn_weights_raw[:, i, i:]
                    attn_weights[:, i, :] = nn.functional.softmax(
                        attn_weights[:, i, :], dim=-1
                    )

                del attn_weights_raw
                del lru_score
                del mean_score
                torch.cuda.empty_cache()

Attention‑based importance

attn_weights has shape (B * H, T_tgt, T_src).

  • batch size
  • head number
  • target length (tgt_len = length of the current query segment (can be shorter, longer, or 1 during decoding))
  • source length (src_len = length of the full key/value history (past_length + current_length)

They sort scores along the source dimension and pick the 0.9 quantile as mean_score, a per‑query/head threshold. For each step i (past start_compress_length), they look at a sliding window [window_start, i) and compute lru_score that marks tokens whose attention ever exceeded the threshold (plus some head aggregation).

Recency bias: They force the last position_window tokens before i to be kept (lru_score = 1), regardless of attention, encoding “recent tokens are always important”. Masking (logical KV drop): For query position i, past keys < i that are not marked important get their attention scores set to a very negative value; after re‑softmax, these tokens get probability ≈ 0.

This means that for this layer, at this query step, only a subset of past tokens (important + recent) are effectively used.

Important nuance: This code does not physically shrink the KV tensors; it still computes full Q/K/V and full attention, then masks.

So it mostly: -Implements the importance selection policy.

  • Measures keep ratios (self.lru_keep_ratio, self.keep_ratio).
  • Demonstrates the quality impact of dropping many tokens.

A more optimized system could use this mask to actually evict KV entries and reduce memory / compute further. After masking, it continues:

  • Applies optional layer_head_mask.
  • Samples attention probs with dropout.
  • Computes attn_output = attn_probs @ value_states.
  • Reshapes back and passes through out_proj.

Personal summary: In training, it will save K,V from back propagation, since it skips unessential hidden states of tokens.

But in inference, it might only saves calculation of V. Since it calculates the full attention weights matrix at first and then filtered it.

And also the dropped token is not propagrated to following layers.
